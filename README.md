# Clinical-Trials-Analysis
This project focuses on analyzing real-world clinical trial datasets to uncover insights into study phases, conditions, interventions, sponsors, enrollment trends, and trial outcomes. Clinical trial data is often complex and unstructured, so the goal of this project is to clean, standardize, explore, and visualize key variables that drive decision-making in pharma and healthcare research.
